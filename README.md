# Community-Board-310-Green-Map
#### All the data is from NYC Open Data. The tree census is a bit old, but I filtered that dataset for living trees (at least, living at the time of the census). The bike paths data is inclusive of all streets that have some infrastructure for biking - whether the street is a designated bike street or has actual designated lanes. Community board boundaries seem like a natural choice for defining a scope of study - neighborhoods often have malleable or ambigious boundaries, and community boards encompass a broader yet still defined atmosphere of a particular area.
